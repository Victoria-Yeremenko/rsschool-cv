Code example

Shortest Word KATA from Codewars: Simple, given a string of words, return the length of the shortest word(s).

String will never be empty and you do not need to account for different data types.

            
              function findShort(s) {
                let newSrt = s.split(' ')
                let len = []
                for (word of newSrt) {
                  len.push(word.length)
                }
                return Math.min(...len)
              }
